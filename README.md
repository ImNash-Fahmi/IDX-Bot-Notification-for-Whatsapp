# Keterbukaan-Informasi
IDX Bot Notification for Whatsapp

1. HMETD
2. PMTHMETD
3. Penyampaian Informasi
4. Transaksi Material
5. Pengumuman RUPS
6. Buyback
7. Pembelian Kembali
8. Restrukturisasi Utang
9. Daftar Efek Bersifat Ekuitas Dalam Pemantauan Khusus
10. Penambahan Modal
